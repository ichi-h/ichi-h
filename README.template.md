## Setup

The development environment is on the [develop](https://github.com/ichi-h/ichi-h/tree/develop) branch.

```sh
git clone -b develop https://github.com/ichi-h/ichi-h.git
cd ichi-h

nix develop
pnpm i

# Develop
pnpm dev

# Build
pnpm build

# Format
pnpm format
```
